# 🚀 HTML Input Playground

Modern HTML input ve form öğelerini interaktif olarak test edebileceğin bir playground.

---

## ✨ Özellikler

* 🎨 Açık / Koyu tema desteği
* 💬 Tooltip (fade + slide + delay animasyon)
* 🔘 Gelişmiş buton davranışları (loading, disabled, ripple vb.)
* 📋 Tek tıkla kod kopyalama
* ⚡ Canlı ve etkileşimli örnekler
* 🧩 Geniş input koleksiyonu (text, color, date, url, vb.)

## 🚀 Nasıl Kullanılır?

### 1. Projeyi indir

```bash
git clone https://github.com/USERNAME/html-input-playground.git
```

### 2. Local server başlat (önerilir)

```bash
cd html-input-playground
py -m http.server 8000
```

### 3. Tarayıcıda aç

```
http://localhost:8000
```

> ⚠️ Not: `file://` ile açarsan bazı özellikler (iframe, clipboard vb.) çalışmayabilir.
