# 🤝 Contributing

Projeye katkıda bulunmak istediğin için teşekkürler!

## 🚀 Nasıl katkıda bulunabilirsin?

* 🐛 Bug bildirimi (Issues)
* 💡 Yeni özellik önerisi
* 🧩 Yeni HTML input örnekleri ekleme
* 🎨 UI/UX geliştirmeleri
* 📝 Dokümantasyon iyileştirme

---

## 📦 Kurulum

```bash
git clone https://github.com/runtoolkit/html-input-playground.git
cd html-input-playground
```

Local server başlat:

```bash
py -m http.server 8000
```

---

## 🛠️ Geliştirme Kuralları

* Kodları **basit ve okunabilir** tut
* Vanilla JS kullan (framework yok)
* Mevcut stil yapısını bozma
* Her yeni örnek için ayrı dosya oluştur (`src/examples/`)
* Gereksiz kütüphane ekleme

---

## 🧪 Test

* Tüm sayfaların çalıştığını kontrol et
* `file://` ve `localhost` farklarını göz önünde bulundur
* Console hatası bırakma

---

## 📝 Commit Mesajları

Önerilen format:

```bash
feat: yeni input örneği eklendi
fix: tooltip bug düzeltildi
style: UI iyileştirmeleri
```

---

## 🔁 Pull Request

* Açıklayıcı başlık yaz
* Ne değiştiğini kısa anlat
* Gerekirse ekran görüntüsü ekle
