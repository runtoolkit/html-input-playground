# 🔐 Security Policy

## 📢 Güvenlik Açığı Bildirme

Bir güvenlik açığı bulduysan:

* ❌ Public issue açma
* ✅ Bunun yerine **GitHub Security Advisories** kullan

GitHub üzerinden:

* "Security" sekmesine git
* "Report a vulnerability" butonuna bas

---

## ⚠️ Kapsam

Bu proje:

* Frontend (HTML/CSS/JavaScript) tabanlıdır
* Sunucu tarafı içermez

Yine de şu riskler önemlidir:

* XSS (Cross-Site Scripting)
* Zararlı script kullanımı
* Güvensiz dış bağlantılar

---

## 🛡️ Güvenlik İlkeleri

* `innerHTML` kullanımından kaçınılır
* Harici scriptler minimumda tutulur
* Kullanıcı girdileri doğrudan işlenmez

---

## ⏱️ Yanıt Süresi

* Güvenlik raporları mümkün olan en kısa sürede incelenir
* Kritik açıklar öncelikli olarak ele alınır
